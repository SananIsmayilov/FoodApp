# FoodApp (Android Bootcamp Final project) 
# Technologies used in this project :
  - MVVM architecture  <br/>
  - Navigation Component <br/>
  - Retrofit2 <br/>
  - Glide <br/>
  - ViewModel , LiveData <br/> 
  - Coroutines <br/>
  - Dagger (Hilt) <br/>
  - Lottie files <br/>
  - ## Databases :
  - Mysql <br/>
  - Room Database <br/>
  - Shared Preferences <br/>
